# docs/legacy

Snapshots históricos del portal **antes del refactor a React + Vite**.

- `Index.html` — versión monolítica original (todo en un HTML, JS inline,
  hojas Google Sheets como backend vía Apps Script). Funcionó como MVP en los
  primeros meses de iPro mientras se validaban los flujos de negocio.
- `index_backup.html` — snapshot intermedio durante la migración.

**No se referencian desde el código en producción.** Quedan acá como
referencia histórica para entender decisiones de diseño que se mantienen en
la app actual (nomenclatura de cajas, layout de la financiera, naming de
módulos, etc.).

Si necesitás replicar un comportamiento que ya estaba en la versión vieja y
no recordás cómo era, abrí estos archivos en el browser local para verlos.
